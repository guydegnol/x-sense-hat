Display Temperature
=====================

将X Sense HAT制作成一个温度计，你可以在RGB点阵或者蓝牙界面的Digital tube上查看温度值。

TIPS
-----

你可以通过此块读取 BME280 检测结果，修改下拉菜单选项，然后选择读取 BME280 检测结果的温度，压强和湿度值。
